## Simulando Uma Conta Bancária Através Do Terminal/Console

Neste projeto, criei um sistema básico de um sistema de cadastramento de contas de banco, seguindo as particularidades que essa conta deve possuir.

# Ferramentas utilizadas

<img src="https://github.com/marwin1991/profile-technology-icons/blob/main/icons/java.png" width="60" height="60" alt="Java">

# Como iniciar?

Para iniciar, é necessario possuir o `Java` instalado, abriremos o terminal na pasta do projeto e executaremos os seguintes comandos:<br>
`javac ContaTerminal.java` -> Para compilar o codigo fonte e assim criar o bytecode java (`.class`).<br>
`java ContaTerminal` -> Para executarmos o arquivo que já foi compilado.<br><br>
Caso de erro, tente atualizar sua versão java para o JDK21, pois foi essa versão utilizada para o desenvolvimento do projeto.<br>

# Como funciona?

O programa pede para o usuário inserir seus dados, utilizando a função `java.util.Scanner` e as variações `nextLine`, `nextDouble` e `nextInt`, é possivel pegar o valor que o usuário digita e registra-lo em uma variável, 
sendo assim possível a criação da conta localmente.
